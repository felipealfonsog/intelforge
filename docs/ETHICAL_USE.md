# Ethical Use (Defensive Only)

IntelForge is intended for defensive cyber threat intelligence:
- authorized collection from public/approved sources
- normalization and provenance tracking
- non-intrusive analysis, signals, and reporting

Prohibited uses include:
- exploitation, intrusion, or unauthorized access
- instructions or automation for harm
- aggressive scanning or targeting guidance

If you are unsure whether a use case is allowed, do not proceed.
